## 🚨 Reporting a Vulnerability

If you discover a security vulnerability, please report it **responsibly** and **privately**.

### ❗ Please Do NOT:

- Open a public GitHub issue
- Share the vulnerability publicly before it is fixed
- Discuss it in community forums or social media

### ✅ Instead, Contact the Maintainer Directly:

| Channel | Details |
|---------|---------|
| 📧 Email | nikhilkumarofficiall123@gmail.com |
| 🔗 GitHub | [@nikhil-ai-insights](https://github.com/nikhil-ai-insights) |

## 🛡️ What to Include in Your Report

To help us resolve the issue as quickly as possible, please include:

```
- Detailed description of the vulnerability
- Steps to reproduce the issue
- Possible impact / severity assessment
- Screenshots or proof-of-concept (if applicable)
- Your suggested fix (optional but appreciated)
```

The more detail you provide, the faster we can assess and address the issue.

## 🔒 Security Best Practices

When using this project, please follow these practices to keep your data safe:

- 🔑 **Keep API keys and credentials secure** — never expose them in your codebase
- 📁 **Do not commit sensitive data** to public repositories
- 🔐 **Use environment variables** instead of hardcoding secrets
- 🚫 **Avoid sharing private Google Sheets publicly**
- 🔄 **Rotate API keys regularly** and revoke any that may have been exposed

### Recommended `.gitignore` entries:

```gitignore
.env
*.key
credentials.json
secrets/
```
